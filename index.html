HTML

Page Start Pattern

<!DOCTYPE html>
    <html>
        <head> helps browser render page, not seen
            <meta charset="utf8"> more details on how to render Page
            <title>Poem</title> (used by browser to determine title of url)
        <style> (for CSS properties)</style>
    </head>
<body> (text that reader will see)</body>
</html>

HTML Text Manipulation

  • Italics: <em></em>
  • Bold: <strong></strong>
  • Strike-through: <del></del>
  • Superscript: <sup></sup>
  • Subscript: <sub></sub>
  • Underline: <ins></ins>

Images

  • <img src="url"> basic image and will show "as is"
  • <img src="url" attribute="text description"> help vision impaired
  • <img src="url" attribute="text description" width="200"> to custom size image, can also use height but not both or lose aspect ratio

Tables

<table>

            <th>column1 header</th>
            <th>column2 header</th>
        </tr>
    </thead>
    <tbody> actual 'data'
        <tr> entering 'data' for row 1
            <td>row1 column1 'data'</td>
            <td>row1 column2 'data'</td>
        </tr>

            <td>row2 column1 'data'</td>
            <td>row2 column2 'data'</td>
        </tr>
    </tbody>
</table>

CSS (Cascading Style Sheets)

CSS rules are contained within the <head> section after the <title>. Use <style></style>
<head>
    <meta charset="utf8">
    <title>Poem</title>
    <style> (for CSS properties)</style>
</head>

Types of Selectors

Selector Types CSS Rule Target ID
Element Selector p {color: red;} all p elements
By ID #rabbit-song {background-color: yellow;} <p id="rabbit-song>
By Class .song-lyrics {color: yellow;} <p class="song-lyrics">

Combining HTML & CSS

HTML and CSS can be combined to affect the appearance of text.
CSS rule: .apples {color: red;}
In body: <strong class="apples">apples</strong>

Font Families

CSS rule for the following paragraph:
p {color:maroon;
text-indent: 50px;
font-family: fantasy;}

This is an example of using CSS and Font Families to change the text of a paragraph. This should be the fantasy 'font'


Specific fonts can be names, but make sure to include the font family in case that particular font does not exist on the client's computer.
For example:
p {font-family: "Helvetica", sans-serif;}, Helvetica is the specific font, sans-serif is the font family.

Font Properties

{font-weight: bold;}, Use to bold sections so that you do not overdo <strong></strong>
{font-style: italic;}, Use to italic section so that you do not overdo <em></em>
{font-size: 13px;}, To size using pixels
{font-size: 1.5em;}, To size using a relative value in relation to the body text size.
{line-height: 1.5em;}, Adjust line height.
{text-align: center;}, To center align text.
{text-decoration: underline;}, To underline beginning text.
